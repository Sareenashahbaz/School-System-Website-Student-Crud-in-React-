# Future Leaders School – Student Admission System

## Project Overview

This project is a **school system website** built using **React.js** and **Bootstrap**. It allows managing student admissions with full **CRUD operations** (Create, Read, Update, Delete). The system stores data in the browser's `localStorage` for persistence.

The project also includes a **Faculty section**, a **Vision statement**, a responsive **Navbar**, and a **Footer**.

---

## Features

### 1. Admission Form

* Collects student details:

  * Full Name
  * Date of Birth
  * Email Address
  * Contact Number
  * Grade Level Applying For
  * Address
* Validates required fields and email format.
* Adds students to the **Applied Candidates Table**.

### 2. Applied Candidates Table

* Displays all students who have applied.
* Supports **CRUD operations**:

  * **Create:** Add new applicants via the admission form.
  * **Read:** View applicants in a responsive table.
  * **Update:** Edit applicant details via modal.
  * **Delete:** Remove applicants from the table.
* Stores all data in **localStorage** for persistence.

### 3. Faculty Section

* Displays teachers with:

  * Full Name
  * Subjects taught
  * Brief bio
  * Image
* Static display (CRUD can be added if needed).

### 4. Vision Statement

* “Our Vision” section highlights the school’s mission and educational values.

### 5. Responsive Layout

* Fully mobile-friendly and responsive on different screen sizes.
* Uses **Bootstrap** for consistent styling and layout.

### 6. Navbar & Footer

* Navbar links to all sections of the website.
* Footer contains dummy contact information and social media icons.

---

## Technologies Used

* **Frontend:** React.js
* **Styling:** Bootstrap 5, custom CSS
* **State Management:** React `useState` and `useEffect`
* **Persistence:** Browser `localStorage`
* **Components:** Functional components and custom modals
